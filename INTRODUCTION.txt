[h1]RimMind - Core[/h1]
让殖民者拥有灵魂的 AI 引擎。RimMind-Core 是整个 RimMind 套件的心脏，接入大语言模型，让你的殖民者不再是冰冷的代码——他们会思考、会感受、会做出属于自己的选择。

[h2] 这是什么 [/h2]
你有没有想过，为什么殖民者只是机械地执行工作优先级？为什么他们不会因为连续加班而崩溃？为什么他们不会在深夜和好友聊天？

RimMind 改变了这一切。

通过接入大语言模型，殖民者获得了真正的"内心世界"：他们会感知周围发生的一切，形成自己的目标，做出基于性格的决策，甚至记住过去的对话和经历。每个殖民者都是独一无二的——因为他们的人格由 AI 实时生成，而非预设脚本。

[h2] 它能做什么 [/h2]

[b]🧠 殖民者有了自己的想法[/b]
殖民者不再只是工作机器。他们会根据当前状态——心情低落？受伤了？刚经历袭击？——形成自己的目标，并决定下一步做什么。一个刚被袭击吓到的殖民者可能会选择躲起来，而不是继续种田。

[b]💬 真正的对话[/b]
点击任意殖民者，和他们聊天。不是从预设列表选选项，而是真正地对话。告诉他们"你今天看起来不太好"，他们会回应自己的感受。每个回复都基于他们当前的状态、记忆和性格。

[b]🎭 独特的人格[/b]
每个殖民者都有 AI 生成的人格——动机、特质、价值观。这些不是随机标签，而是真正影响他们决策的内在驱动力。一个重视"家庭"的殖民者会更关心亲人的安危；一个追求"权力"的殖民者可能会主动争取领导地位。

[b]📖 记忆与成长[/b]
殖民者会记住发生的事情——那场惨烈的袭击、和好友的深夜长谈、第一次击杀敌人。这些记忆会影响他们后续的想法和行为，让角色随时间成长。

[h2] 技术亮点 [/h2]

[b]智能上下文[/b] — 自动采集游戏状态为 AI 提供完整情报：殖民者心情、健康、技能、装备、地图财富、威胁等级、季节天气……AI 看到的和你一样多。

[b]零卡顿[/b] — 所有 AI 请求在后台线程执行，主线程只处理结果。你的游戏帧率不会受到任何影响。

[b]多模型支持[/b] — 兼容 OpenAI / DeepSeek / Ollama 等所有 OpenAI 格式的 API，也支持 Player2 本地服务。用云端模型获得最佳效果，或用本地模型保护隐私。

[b]Token 预算管理[/b] — 智能裁剪上下文，在信息量和成本之间自动平衡。提供最小/标准/完整三种预设，也可自定义 28+ 个选项。

[b]数据飞轮[/b] — 内置自动调优系统，持续分析 AI 请求效果并优化参数，越用越智能。

[h2] 快速开始 [/h2]

[b]1. 安装[/b]
从 Steam 创意工坊安装 RimMind-Core 和 Harmony 前置。

[b]2. 配置 API[/b]
进入 选项 → 模组设置 → RimMind-Core：
[list]
[*] 选择 AI Provider（推荐 DeepSeek，性价比最高）
[*] 填入 API Key
[*] 点击"测试连接"确认
[/list]

[b]3. 开始游玩[/b]
安装任意子模组即可体验。推荐从 RimMind-Personality 开始，看看殖民者都在想什么。

[h2] 常见问题 [/h2]

[b] Q: 会不会卡？ [/b]
A: 不会。所有 AI 请求在后台执行，游戏完全不会卡顿。

[b] Q: API Key 安全吗？ [/b]
A: 仅存储在本地，只发送给你配置的 API 端点，不会上传到 RimMind 服务器。

[b] Q: 不想花钱怎么办？ [/b]
A: 用 Ollama 跑本地模型，完全免费。或者用 DeepSeek，价格极低。

[b] Q: 只装 Core 不装子模组会怎样？ [/b]
A: Core 本身提供基础设施，不装子模组看不到明显效果。推荐至少装一个 Personality 或 Dialogue。
